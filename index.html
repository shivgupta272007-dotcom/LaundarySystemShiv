🧺 LaundryFlow System

This is a full-stack Node.js application built for the Mini Laundry Order Management System assignment.

Because this application requires a NodeJS backend server and database routing to function, the live app cannot be fully automated directly via static GitHub pages.

To evaluate the assignment with the dynamic UI, Dashboard, and API routes running, please clone the repository and run the app locally:

npm install
npm start

The app will be available immediately at http://localhost:5000
